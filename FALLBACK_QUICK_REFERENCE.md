# Limited Analysis Mode - Quick Reference Card

## 🚨 What You'll See

When satellite imagery is unavailable, you'll see these indicators:

### 1. Toast Notification (Top-Right)
```
⚠️ Analysis completed using NDVI data only
```

### 2. Warning Message
```
⚠️ Analysis completed in X.XXs (Limited mode: Satellite imagery unavailable)
```

### 3. Error Banner (Red Box)
```
🚨 IMPORTANT: Limited Analysis Mode Active
Satellite imagery unavailable - Analysis based on NDVI only
```

### 4. Pipeline Status
```
AI Analysis: ⚠️ Partial (instead of ✅ Complete)
```

### 5. Confidence Label
```
Confidence: 75% (NDVI-only) ← Note the suffix
```

## ✅ What Still Works

- ✅ NDVI health assessment (accurate)
- ✅ Risk classification (reliable)
- ✅ Recommendations (valid)
- ✅ Plot registration (normal)
- ✅ Alert creation (normal)

## ⚠️ What's Limited

- ⚠️ No satellite image display
- ⚠️ No visual verification
- ⚠️ Slightly lower confidence (70-85% vs 90-95%)

## 🤔 Why Does This Happen?

- Recent imagery not yet available
- Cloud cover during satellite passes
- Location outside current coverage
- Data processing delays

**This is normal!** Happens for many locations.

## 💡 What Should You Do?

### If Risk is LOW or MEDIUM
- ✅ Continue normal monitoring
- ✅ Follow recommendations
- ✅ No immediate action needed

### If Risk is HIGH or CRITICAL
- 🚨 Take action immediately!
- 🚨 Inspect your field
- 🚨 Follow recommendations
- 🚨 Contact extension officer if needed

**Don't wait for imagery!** NDVI data is reliable.

## 📊 Confidence Scores

| Mode | Confidence | Reliability |
|------|-----------|-------------|
| Full | 90-95% | Excellent |
| Limited | 70-85% | Very Good |

Both are reliable for crop monitoring!

## ❓ Quick FAQ

**Q: Is the analysis still accurate?**
A: YES! NDVI is proven and reliable.

**Q: Should I wait for imagery?**
A: NO! Act on the risk level shown.

**Q: Will imagery be available later?**
A: Possibly. Try again in a few days.

**Q: Can I trust NDVI-only analysis?**
A: YES! It's used worldwide for crop monitoring.
